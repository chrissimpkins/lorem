Lorem

A lorem ipsum dummy text generator that inserts or appends dummy text in existing files, creates new text files, or writes the text to the standard out stream.

Download this project as a .zip file Download this project as a tar.gz file

Installation

Download the source from one of the links to the right above. Unpack the source and open your terminal in the main source directory. Then enter the following on the command line:

./configure
make
sudo make install

If you get an error, you likely need to update GNU Automake on your system. You can download and install updated packages through your package manager or on the GNU ftp site.

You can confirm that it is installed by checking the help with the following command:

lorem -h

If it worked, it works! Now you can generate more dummy text than you ever thought possible!

If all else fails, you can build the source with the following command (assuming that you have gcc):

gcc lorem.c -O3 -Wall -g -o lorem

Manually move the lorem binary into a directory on your PATH or use it from the directory where you installed it like this:

./lorem

Use Examples

Create a New File with Dummy Text

Say you want to create a new file named 'eg.txt' with 5 paragraphs of dummy text. Use the following command:

lorem -n 5 -o eg.txt

Append Dummy Text to an Existing Text File

You have a file named 'eg.txt' and you would like to append 10 paragraphs of dummy text to it. Use the following command:

lorem -a -n 10 -o eg.txt

Pipe Dummy Text to Another Application

lorem will write up to one million paragraphs of dummy text to the standard out stream. That's over 69 million words in Latin (>467 MiB)! Pipe it to whatever you like.

lorem -s -n 1000000 | wc -c

lorem -h
